LATEST WORK

Title ： Multi vs Multi Keidoro with MTS
制作年 : 2017
開発言語 : Java
コメント ：
多対多のケイドロを行うアプリケーションです，緑色の泥棒アイコンがプレイヤーで，上下左右の矢印キーで移動することができます． この迷路の地形はアプリケーション起動時に毎回ランダムに生成され，上下左右の端は繋がっており， 自由に行き来することができます．
